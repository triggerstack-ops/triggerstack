n8n 2.10.3 or higher. If you use n8n Cloud, you're already compatible. If self-hosting, run the latest Docker image. These workflows will NOT work on n8n 1.x.
Do I need coding experience?
Zero. Every workflow is pre-built \u2014 just import the JSON files and follow the setup guide. If you can copy an API key, you can run this.
What are the ongoing costs?
$0/month. n8n (self-hosted = free), SerpAPI (100 searches/month free), Google Sheets (free), Gmail SMTP (500 emails/day free). You only pay if you outgrow free tiers.

~400+ per month. SerpAPI gives 100 free searches = 20 full 5-platform scrapes. Each returns 20+ leads. EnrichIQ and OutreachEngine are unlimited.

The others keep running. Every scraper has error handling so one failing platform never kills the whole workflow. This is a major advantage over competing templates.

Absolutely. Many buyers use this to run lead gen for multiple clients. Just duplicate the Google Sheet per client and adjust the search queries. One n8n instance handles it all.
